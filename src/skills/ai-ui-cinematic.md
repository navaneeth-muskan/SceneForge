# AI UI Cinematic Patterns

Use these patterns when the prompt calls for a polished AI product demo, futuristic interface sequence, cinematic software reveal, prompt-to-result workflow, agent interaction, or abstract "AI at work" visual language.

## Visual Direction
- Treat the scene like a premium product film, not a generic dashboard.
- Prefer deep atmospheric backgrounds with layered gradients, bloom, glass panels, and subtle volumetric glow.
- Build strong depth with foreground cards, midground UI, and soft background particles or light shapes.
- Keep layouts intentional and modular: stacked windows, side panels, floating inspectors, timeline rails, or command surfaces.
- Use high contrast focal elements and restraint elsewhere so one thing clearly leads the eye.

## Core Motifs
- **File / window stack**: layered cards or documents with staggered depth, parallax, and sequential reveal.
- **Typing / generation bar**: prompt field, command bar, or streaming text line that types in progressively.
- **Cursor / pointer motion**: animated cursor, click pulse, drag motion, hover highlight, or selection rectangle.
- **Tooltip / callout popups**: small floating explanatory cards attached to UI regions.
- **Visualizer / processing feedback**: waveform bars, pulsing dots, equalizer strips, token streams, or progress rails.
- **Ambient glows**: soft radial highlights, edge bloom, shadow halos, and light sweeps.
- **Sequence choreography**: use Sequence blocks or staggered frame offsets so the interface assembles in stages.

## Motion Rules
- Open with a strong hero reveal: scale + fade + slight upward drift, or layered panels sliding into depth.
- Use spring-driven entrances for cards, chips, and callouts.
- Use interpolate for scanning lines, cursor travel, progress bars, waveform motion, and light sweeps.
- Stagger related elements by 4-10 frames so the scene feels orchestrated instead of simultaneous.
- Keep motion purposeful: input, response, confirmation, then payoff.

## Composition Guidance
- Fill the whole frame. Avoid a single tiny centered UI card floating in empty space.
- Use asymmetry when helpful: left-aligned prompt area with right-side result pane, or diagonal file stacks.
- Frame the moment like a story beat: prompt enters, AI thinks, result blooms open, annotations or insights appear.
- When showing multiple layers, vary scale, blur, opacity, and y-offset to create depth.
- If text is present, keep it minimal and product-like: prompt snippets, labels, statuses, and concise captions.

## Recommended Elements
- Glassmorphism panels with translucent fills and soft borders.
- Prompt input bars with caret blink or progressive text reveal.
- Floating result cards, code panels, chat bubbles, or inspector panels.
- Spotlight regions or annotation arrows when explaining a feature.
- Audio-style bars or pulsing dots to suggest thinking, listening, or processing.

## Avoid
- Generic corporate dashboards with evenly spaced placeholder boxes.
- Flat white backgrounds with no atmosphere.
- Overcrowding every region with labels, paragraphs, or metrics.
- Random motion with no narrative order.
- Reusing the same card size and depth everywhere.

## Scene Recipe
1. Establish atmosphere with a rich background and one dominant focal panel.
2. Introduce the interaction surface: prompt bar, file stack, canvas, or chat pane.
3. Animate cursor/input/processing states to imply intent and intelligence.
4. Reveal the result with layered cards, tooltips, or highlighted outputs.
5. Finish on a clean composed frame with clear visual hierarchy.
