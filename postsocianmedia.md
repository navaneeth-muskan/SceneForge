# Social Media Post (Hackathon Bonus)

I built **SceneForge** with **Muskan**: an agentic multimodal motion-graphics studio powered by Google AI models and Google Cloud.

## How It Works

1. User prompt + multimodal inputs (YouTube URLs, website URLs, PDFs/images/audio/video)
2. Gemini-powered analysis and planning
3. Structured outputs into scene/timeline generation
4. Remotion live preview workflow
5. Cloud-backed media pipeline using Google Cloud services

## Tech Highlights

1. Google AI: Gemini models for multimodal understanding, generation, and narration workflows
2. Google Cloud: deployment + media/storage infrastructure
3. Agentic pipeline for scene planning and build orchestration
